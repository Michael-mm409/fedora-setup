# 🚀 Michael's Fedora Setup (Masters of Data Science)

This repository contains the "Gold Master" configuration for Michael's Fedora 43 environment, optimized for a Desktop (NVIDIA/Proxmox) and an IdeaPad Laptop (Battery-Safe Sync).

## 🛠️ Quick Start (New Machine)
1. Install Fedora 43 KDE.
2. Install and run `sudo tailscale up`.
3. Clone and Execute:
   ```bash
   git clone git@github.com:Michael-mm409/fedora-setup.git ~/fedora-setup
   cd ~/fedora-setup && chmod +x setup-fedora.sh && ./setup-fedora.sh
   source ~/.bashrc
   ```
